uint32_t SDFindEmptySpace ( const uint8_t  restore)

Find the first empty allocation unit in the FAT.

The value of the first empty allocation unit is returned and its location will contain the end-of-chain marker, SD_EOC_END. NOTE: It is important to realize that, though the new entry now contains an EOC marker, this function does not know what cluster is being extended and therefore the calling function must modify the previous EOC to contain the return value

Parameters:
restoreIf non-zero, the original fat-sector will be restored to g_sd_fat before returning; if zero, the last-used sector will remain loaded
Returns:
Returns the number of the first unused allocation unit
