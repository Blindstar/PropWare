Detailed Description

Author:
David Zemon, Collin Winans

MCP300x analog-to-digital driver for Parallax Propeller using SPI communication. Supports both MCP3004 and MCP3008 devices along with both single-ended and differential readings.

Note:
MCP300x chips uses SPI mode 2 and shifts data MSB first
