uint8_t SPIGetPinNum ( const uint32_t  pinMask)

Retrieve the pin number from a pin mask; i.e., if pinMask is 0x01, return 0; if pinMask is 0x40, return 6.

Precondition:
Only 1 bit is set in pinMask (if more than one is set, the return value will be related to the least significant set bit)
Parameters:
pinMaskThe bit number of the set bit in this variable will be returned
Returns:
Returns the pin number of pinMask (no error checking)
